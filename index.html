Press A Key to Start

How to play the game

1. Start the Simon game, and it will begin with a sequence of one color and sound.

2. Repeat the sequence by pressing the corresponding color buttons in the same order as the game presented them.

3. If you correctly repeat the sequence, the game will add one more color and sound to the sequence and play it back to you.

4. Continue to repeat the sequences, adding one more color and sound each time, until you make a mistake.
